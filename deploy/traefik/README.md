- select a special chart version to install ,defalut by latest
helm pull traefik/traefik --version <version>
- create a value-prod.yaml write configuration,overriding the default values.yaml
- install traefik
  ``` shell 
  helm install --namespace kube-system traefik ./traefik -f ./traefik/values-prod.yaml
  ```
